### 守护进程supervisord
```
安装
pip install supervisor

编辑配置：
vim /etc/supervisord.conf
最后一行设置了配置文件的目录
[include]
files = supervisord.d/*.ini

进入/etc/supervisord.d
创建守护进程的配置文件
[program:lumen-clsbiz-worker]
process_name=%(program_name)s_%(process_num)02d
command=php /data/site/public_clsbiz_api/artisan queue:work --sleep=3 --tries=3 --daemon
autostart=true
autorestart=true
user=root
numprocs=1
redirect_stderr=true
stdout_logfile=/data/logs/public_clsbiz_api/worker.log

载入配置
supervisorctl reread
supervisorctl update
重启进程
supervisorctl restart all
帮助
supervisorctl help

laravel 重新启动job队列进程
php artisan queue:restart
```
