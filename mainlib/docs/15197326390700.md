# crontab
~~~txt
cron日志 tail -f /var/log/vron.log

crontab -l 任务列表
crontab -e 编辑任务
crontab -r 删除任务

ps -A | grep cron cron进程 kill -9 pid

rm -rf /var/run/crond.pid 删除所有cron进程

cron restart 重启cron

service cron stop 关闭
~~~
